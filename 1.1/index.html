aprsworld-to-XML Interface
The scripts on this site provide XML interfaces to aprsworld.net, so you don't need to have direct access to the database, or know SQL, in order to get generalized and standardly formatted APRS data directly from the Internet into your application. Free code libraries for parsing XML are easy to find for almost any programming environment.

This script is geared to the specific uses that I have for it, so if something is lacking that you could make use of, just let me know and I'll try to add or change whatever it is. I'd also be curious to hear from anybody else who is able to make use of this site.

The most recent version of the findU-to-XML interface can be found here, but the findU backup server that it gets its data from was still not working as of this writing.

The data is provided by aprsworld.net by Jim Jarvis KB0THN. APRS was created by, and is a trademark of, Bob Bruninga WB4APR.
