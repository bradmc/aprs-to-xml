findU-to-XML Interface
The backup findu server is down, so this XML-formatted data is currently unavailable. No word on when it will return.

The scripts on this site provide XML interfaces to findu.com, so you don't need to have direct access to the database, or know SQL, in order to get generalized and standardly formatted APRS data directly from the Internet into your application. Free code libraries for parsing XML are easy to find for almost any programming environment.

These scripts are geared to the specific uses that I have for them, so if something is lacking that you could make use of, just let me know and I'll try to add or change whatever it is. I'd also be curious to hear from anybody else who is able to make use of this site.

The data is provided by findu.com, created by Steve Dimse K4HG. The linked maps are at aprsworld.net by Jim Jarvis KB0THN. APRS was created by, and is a trademark of, Bob Bruninga WB4APR.

Get a Weather Report

http://xml.n8qq.com/aprs/1.0/findwx.cgi?call=n8qq

Parameters:
  • call = station callsign

Try it ... call
Multiple Position Reports

http://xml.n8qq.com/aprs/1.0/posit.cgi?call=n8qq&start=5&length=3

Parameters:
  • call = station callsign
  • start = number of hours back to start (default: 24)
  • length = number of hours to display (default: start time)

Try it ... call start length
